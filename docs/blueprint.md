# **App Name**: ASI Portal

## Core Features:

- Authentication & User Management: Secure email/password authentication with Firebase Auth, auto-role assignment, role-based routing, and user profiles with company association.
- Job Lifecycle Management (JLM): Job cards with auto-generated numbers, multi-vehicle support, damage documentation, quote generation, status tracking, technician assignment, and job-specific permissions.
- Booking Hub: Direct booking form for repairs, inspections, manufacturing services, films installation, vehicle information capture, damage assessment, and preferred scheduling.
- CRM & Sales Pipeline: 6-stage pipeline, lead management, sales activities tracking, sales tasks, contact organizations, organization contacts, CSV bulk import, drag-and-drop pipeline, value tracking and probability scoring.
- Google Calendar Integration: Auto-sync to user's Google Calendar, day/week/month views, event creation from job bookings, technician scheduling, bidirectional event updates, pending invites display, and Calendar API integration with OAuth2.
- AI Assistant Integration: Integration of Relevance AI for specialized agents (HECTOR for sales, ATLAS for project management), Universal Firebase REST API, agent-specific prompts, and cross-agent collaboration using appropriate tools for context and information retrieval.
- Inspection System: Multi-vehicle inspection reports, detailed damage documentation per vehicle with photo uploads, client approval workflow, inspection to job conversion.

## Style Guidelines:

- Primary color: A saturated violet (#8000FF) to give a sense of modernity and sophistication.
- Background color: Dark grayish violet (#262633), appropriate for a dark color scheme and professional style.
- Accent color: Blue (#0080FF), used to bring some highlights, such as in call-to-action buttons.
- Body font: 'Inter', a sans-serif font known for its neutral and readable design.
- Headline font: 'Space Grotesk', a geometric sans-serif to differentiate titles from body text with its unique characteristics.
- Code font: 'Source Code Pro' for displaying code snippets.
- Card-based layouts with glass morphism.
- Mobile-first responsive design to support different screen sizes.
- Toast notifications and Modal dialogs for user feedback and interactions
